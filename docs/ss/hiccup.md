# Hiccup 打嗝

### Introduction

- Definition：急性打嗝：<48 hrs；慢性打嗝：>48 hrs；頑固性打嗝：> 1 month

### Differential diagnosis

| 分類 | 疾病 / 狀況 |
|------|-------------|
| 中樞性 | brain tumor、CVA、Parkinson’s disease、multiple sclerosis |
| 器質性 | chest / abdomen / GI disease（如 GERD） |
| 代謝性 | hyponatremia、hypokalemia、hypocalcemia、hypobicarbonate、hyperuricemia、alcohol consumption |
| 系統性 | chest / abdomen / GI infection |
| 心因性 | stress、anxiety |
| 醫源性 | brain / chest / abdomen surgery、endotracheal tube、drugs（dopamine agonist、opioids、BZDs、barbiturates、steroids） |

### Initial management

- 非藥物治療
    - valsalva maneuver、屏息、拉舌頭、冰水敷臉或攝入
- 藥物治療
    - Central：baclofen、gabapentin
    - non-central：PPI、metoclopramide、baclofen
