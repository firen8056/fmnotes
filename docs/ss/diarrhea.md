# Diarrhea 腹瀉

### Introduction

- Definition：<4W：Acute diarrhe、>4W：chronic diarrhea

### Differential diagnosis

**Acute diarrhea**

| 類別 | 次分類 | 常見原因 |
|------|---------|-----------|
| Infection | Viral | Rotavirus、Norovirus、CMV |
|            | Bacterial | E.coli、Vibrio、Campylobacter、Salmonella、Shigella、Clostridium difficile、Yersinia、Aeromonas、Listeria |
|            | Parasitic | Giardia、Cryptosporidia、Cyclospora、Entamoeba histolytica |
| Medication | — | antibiotics、laxatives、magnesium-based antacids、colchicine、SSRIs |

**Chronic diarrhea**

| 類別 | 分類依據 / 特徵 | 次分類 | 常見原因 |
|------|------------------|---------|-----------|
| Watery diarrhea | check stool osmotic gap（290 - 2（stoolNa + stoolK）） | Osmotic（fecal fat−、osmotic gap >100–125、↓ with fasting） | lactose intolerance、celiac disease、magnesium-containing antacids、phosphates、mannitol、sorbitol |
|  |  | Secretory（normal osmotic gap、no change with fasting） | Endocrine：hyperthyroidism、Addison’s disease；GI tract neoplasm：carcinoma、lymphoma、villous adenoma；Neuroendocrine tumor：VIPoma、carcinoid；Microscopic colitis、bile acid malabsorption |
|  |  | Functional（normal osmotic gap、↓ with fasting） | Irritable bowel syndrome、functional diarrhea |
| Fatty diarrhea（malabsorption / maldigestion） | fecal fat+、↑ osmotic gap、↓ diarrhea with fasting | — | pancreatic insufficiency、celiac disease、lactose intolerance、bile acid malabsorption、hepatobiliary disorder、small intestinal bacterial overgrowth、Whipple’s disease、short bowel syndrome、radiation enteritis、mesenteric ischemia |
| Inflammatory | fecal WBC+、calprotectin+、FOB+、fever、abd pain | — | Infection：C.diff、Entamoeba histolytica、Yersinia、CMV、TB；Inflammatory bowel disease：CD、UC；Others：ischemic colitis、neoplasm |

### Approach

- Acute diarrhea
    - 病史
        - 症狀本身：onset、stool features、frequency and volume、PO intake
        - 合併症狀：inflammatory (fever、abd pain、tenesmus、nausea or vomiting、bloody or pus in stool)、hypovolemia (thirsty、decreased UOP、orthostasis)、myalgia
        - 過去病史：immunosuppression、drugs、recent abx
        - TOCC
    - PE
        - vital signs、fluid status (mucosa、JVP、skin turmor)
        - GI：bowel sound、tenderness、distension
        - Skin：jaundice、rashes
    - 檢查
        - Lab
            - CBC、BUN、Cre、Na、K、Ca、Cl、blood culture
            - Stool：WBC、OB、culture、ova+parasite、C.diff、specific culture、calprotectin
            - urinalysis
        - Image：KUB or CT if severe abd pain or concern for toxic megacolon
- Chronic diarrhea
    - 病史
        - 症狀：發生時間、糞便特徵、frequency、exacerbating factors、有無間歇性便祕、禁食的影響
        - 合併症狀：abd pain、weight loss、incontinence、hyperthyroid symptoms
        - 過去病史：thyroid disease、previous radiotherapy、OP、pancreatitis
        - 物質等：drugs、toxins、TOCC、abx
    - PE
        - BMI、vital signs、fluid status、lymph nodes
        - 腹部：bowel sound、distention、tenderness、masses
        - Rectal：fistula、sphincter tone、stool impaction
    - 檢查
        - Lab
            - Blood：CBC、BUN、Cre、Na、K、Ca、Cl、albumin、LFT、TSH、ESR、anti-TTG/total IgA
            - stool：WBC、OB、pH、culture、ova+parasite、C.diff、calprotectin、fat、 stool osmolar gap (Na、K)
        - Colonoscopy
