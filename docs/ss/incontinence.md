# Urinary incontinence 尿失禁

### Differential diagnosis

| 分類 | 次分類 | 疾病 / 狀況 |
|------|--------|-------------|
| Stress incontinence | — | estrogen deficiency、NSD、pelvic surgery、chronic UTI |
| Urge incontinence | Neurologic | CVA、SCI、multiple sclerosis、Parkinson disease |
|                    | Irritation | infection、stone、tumor |
|                    | Obstructive | BPH |
|                    | Idiopathic | OAB |
| Overflow incontinence | Structure | BPH、DM cystopathy、urethral stricture |
|                        | Neurologic | DSD |
| Functional incontinence | — | DIAPPERS（delirium、infection、atrophic vaginitis、pharmacologic、psychiatric、endocrine、restricted mobility、stool impaction） |


### Approach

- Hx
    - symptom and course：trigger、時間、頻率、嚴重性
    - 合併症狀：dysuria、hematuria、夜尿、頻尿、急迫感、咳嗽漏尿
    - 其他合併症狀：sexual function、bowel function
    - 婦產科病史、過去病史、手術史、目前用藥、飲水量以及咖啡因攝取
    - Voiding diary
- PE
    - mobility
    - abdomen：腫塊、壓痛
    - Pelvis：prolapse、pelvic muscle control、infection or inflammation、atrophy
    - DRE：mass、stool impaction
- Exam
    - Initial
        - Lab：UA、Ucx、Ucytology、chem-7（Na-K-Cl-CO2-BUN-crea-Glu）、HbA1c、B12
        - Image：Bladder ultrasound、PVR
    - Others
        - cough stress test、pad test、Q tip test
        - 尿路動力學（若考慮手術）    

### Initial management

- Urge：膀胱訓練、棉墊、骨盆底肌肉訓練、藥物（anti-muscarinic、beta-3 agonist）
- overflow：治療 BPH
- stress：膀胱訓練、棉墊、骨盆底肌肉訓練、藥物（alpha adrenergic、cymbalta、荷爾蒙軟膏）
