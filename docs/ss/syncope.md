# Syncope 昏厥

### Differential diagnosis

| 分類                | 次分類 / 疾病 | 特徵 / 說明 |
|---------------------|---------------|-------------|
| Neurally-mediated   | Vasovagal     | 有 aura（噁心、冒汗、視力模糊、頭暈），常因站太久或不適狀況引發 |
|                     | Situational   | 常因咳嗽、用力或 valsalva maneuver |
| Orthostatic hypotension | Hypovolemia   | — |
|                     | Vasodilation  | — |
|                     | Dysautonomia  | Parkinson、Lewy body dementia、POTS、DM、CKD、喝酒、amyloidosis |
| Cardiovascular      | Arrhythmia    | — |
|                     | Mechanical    | Valvular disease、HCMP、pericardial temponade、PE、dissection、ruptured AAA |
| Neurologic          | —             | CVA、TIA、SAH、cerebrovascular dissection、vertebrobasilar insufficiency、migraine |
| 其他 LOC 原因       | —             | 低血糖、癲癇、缺氧、藥物、psychogenic |


### Approach

- Hx
    - 症狀：過去有無、在做甚麼、誘發因子（咳嗽、打噴嚏、疼痛、便祕、換姿勢、轉頭）、有無 trauma
    - 合併症狀：胸痛、喘、心悸、神經學症狀、prodrome、postictal
    - PH：過去有無心臟或神經病史
    - 家族病史：CMP、sudden cardiac death、syncope
    - 物質藥物史
- PE
    - orthostatic BP、上下肢、左右手 BP、heart sound、lung exam、NE、carotid bruits、volume status
    - 若有 Trauma，檢查受傷部位
- Exam
    - Lab：CBC、alb / TP、AST / ALP、Glu / HbA1c、cholesterol、BUN / cre、hCG、INR if warfarin、urinalysis / urine culture if elderly
        - 若病史懷疑可考慮 B12、AM cortisol、syphilis、HIV、BNP、troponin、D-dimer；24hr urine Na
    - Heart：ECG、holter、echo、EP studies
    - Neuro：head CT、EEG
    - Other：tilting table
