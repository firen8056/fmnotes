# Abdominal Pain 腹痛

### Introduction

- Definition
    - Acute abdominal pain：數天，常 < 7天，或很嚴重也可稱急性
    - Chronic abdominal pain：常 > 3-6個月

### Differential diagnosis

**Acute Abdominal Pain**

| 區域 / 類別 | 可能原因 |
|--------------|-----------|
| RUQ | cholecystitis、pneumonia、APN |
| Epigastric | MI、aneurysm、GERD、PUD、cholecystitis、pancreatitis |
| LUQ | pneumonia、APN、PUD、pancreatitis、splenic infarct |
| Periumbilical | appendicitis、SBO、AAA、DKA、mesenteric ischemia |
| RLQ | appendicitis、IBD、hernia、APN、OBGYN |
| LLQ | diverticulitis、hernia、APN、OBGYN |
| Pelvic | IBD、diverticulitis、UTI、OBGYN |


**Chronic Abdominal Pain**

| 類別 | 可能原因 |
|------|-----------|
| Epigastric | dyspepsia、GERD、PUD |
| Inflammatory | IBD、celiac disease、chronic pancreatitis |
| Motility | constipation、gastroparesis |
| Vascular | chronic mesenteric ischemia |
| Neoplasm | CRC、HCC、gastric CA、pancreatobiliary CA、ovarian CA |
| Functional | IBS、functional abdominal pain syndrome、functional dyspepsia、functional constipation |
| Gynecology | Gynecology etiology |


### Approach

- Hx
    - 疼痛：LQQOPERA
    - 合併症狀：nausea or vomiting、constipation or diarrhea、rectal bleeding、reflux、no flatus、abd distention、fever、edema、jaundice、weight loss
    - 過去病史：cancer、CAD、PAD、abd OP、immunocompromised、endocrinopathy、LMP、sex
    - 藥物物質：NSAIDs、abx、菸酒、travel
- PE
    - General：vital signs、appearance
    - Abdomen：skin、bruits、bowel sound、distension、tenderness、guarding、rebounding、mass、megaly、percussion
        - Special signs：Murphy signs、Psoas signs、LAP
- Exam
    - Lab
        - Acute
            - Blood：CBC、BMP (Glu、BUN、cre、Na / K / Ca / Cl / CO2)、LFT、lipase、lactate、hCG
            - Urine：urinalysis
        - chronic
            - Blood：CBC、BMP (Glu、BUN、cre、Na / K / Ca / Cl / CO2)、LFT、lipase、TSH、iron profile、celiac testing
            - Urine：urinalysis
            - Stool：FOBT
    - Image：x ray、CT、ultrasound
    - 若懷疑可做胃鏡、大腸鏡
