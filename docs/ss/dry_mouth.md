# Dry mouth 口乾

### Differential diagnosis

| 分類 | 次分類 | 疾病 / 狀況 |
|------|--------|-------------|
| Local | — | mouth breathing |
|       | Salivary gland disease | stone |
|       | Local infection | HFM disease、chickenpox、HSV、periodontal disease |
|       | Local irritation | radiation、tobacco、denture |
|       | Local malignancy | — |
| Systemic | Inflammation | Sjogren syndrome、RA、SLE、scleroderma、sarcoidosis |
|           | Infection | HCV、CMV、EBV、HIV |
|           | Endocrine and metabolic | DM、hyperthyroidism、hypothyroidism、dehydration、CKD、anemia |
|           | Neurologic or psychologic | Parkinson disease、cognitive alteration、depression、anxiety、fibromyalgia、eating disorders |
| Drug | — | anti-HTN、antispasm、TCA、anticholinergic、antihistamine、alcohol 等 |


### Approach

- Hx
    - 症狀：吃東西要喝水嗎、吃東西會口乾嗎、吞東西困難嗎、覺得口水量太少嗎
    - 可疑病史的合併症狀
    - 過去病史：AIR、infection、endocrine (DM、thyroid、CKD、Parkinson)
    - 物質治療：手術、RT、藥物、抽菸、喝酒
- PE
    - 口腔外：salivary gland、lymph node
    - 口腔內：soft tissue、periodontium、dentition
- Exam
    - Lab：CBCDC、IgG-A-M、ESR、ANA、antiSSA or antiSSB、RF、thyroid function、iron、zinc、B12/folate
    - Image：Scintigraphy、echo、MRI、CT等
    - salivary biopsy

### Initial mangement

- 非藥物治療
    - 多吃
        - 喝水、低糖飲料。
        - 低糖口香糖、低糖硬糖果 → 可促進唾液流動
        - 漱口水 → 避免蛀牙
    - 不吃
        - 咖啡因、菸草、酒精 → 會讓口乾
        - 辣、鹹 → 可能會口痛
    - 口腔潤濕劑，如唾液替代物。使用專門製造改善口乾的漱口水，對口乾也很有效。
- 藥物
    - 促進口水：pilocarpine 5-10mg TID
    - 感染：Chlorhexidine for candida、abx 7-10 days for bacteria
