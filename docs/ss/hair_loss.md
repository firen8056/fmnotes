# Hair loss 掉髮

### Introduction

- Definition：每天正常掉髮為 100 根。若 >100 根就有可能是異常掉髮
- Classification
    - diffuse vs non-diffuse
    - scaring vs non-scaring

### Differential diagnosis

| 分類 | 疾病 / 狀況 | 說明 |
|------|-------------|------|
| Structural abnormal | Trichorrhexis nodosa | 角化異常 |
|                      | Pili torti | 先天造成頭髮扭曲 |
| Androgen related | Androgenetic alopecia | — |
| Autoimmune related | Alopecia areata | — |
| Cycle abnormal | Telogen effluvium | — |
|                 | Anagen effluvium | — |
| Infection related | Tinea capitis | — |
|                   | Secondary syphilis | — |
| Trauma related | Trichotillomania | — |
|                 | Traction alopecia | — |
| Other | Scarring alopecia | — |


### Approach

- PE
    - hair pull test
        - 若 >10% 以上頭髮被拉扯掉則為陽性，代表會主動掉髮
        - 在顯微鏡下看是否完整、是否在生長期（生長期有sheath）、黴菌感染
    - Hair shedding count
- Exam
    - Lab：TSH、ANA、VDRL、Hb、liver function、ferritin、zinc 看營養狀況
        - 女性禿髮若還有多毛、月經異常、不孕等，可驗荷爾蒙：prolactin、testosterone、DHEA 等
    - KOH、Wood’s light for tinea capitis
    - Biopsy
    - phototrichogram
    - Trichoscopy
