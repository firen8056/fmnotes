# Weight loss 體重減輕

### Introduction
- Definition：6 個月內體重減輕 >5%

### Differential diagnosis

**食量上升**

| 分類       | 原因 |
|------------|-------------|
| 生活型態    | 運動量增加 |
| 內分泌     | uncontrolled DM、hyperthyroidism |
| 吸收異常   | Malabsorption syndrome |


**食量下降**

| 分類             | 原因 |
|------------------|-------------|
| 惡性腫瘤         | **惡性腫瘤（最大宗原因）** |
| 腸胃道疾病       | oral ulcer、dysphagia、peptic ulcer disease、malabsorption、inflammatory bowel disease |
| 精神疾病         | depression、bipolar disease、anxiety、厭食症 |
| 內分泌           | DM、甲狀腺低下、副甲狀腺亢進、pheochromocytoma |
| 慢性感染         | TB、fungus、infectious endocarditis、HIV |
| 慢性發炎         | SLE、RA、Scleroderma、Sarcoidosis、Giant cell arteritis |
| 神經系統疾病     | Stroke、dementia、Parkinson disease |
| 其他系統性疾病   | heart failure、COPD、CKD |
| 藥物             | 某些糖尿病用藥、抗癲癇藥、SSRI、甲狀腺素等 |

### Approach

- Hx
    - 營養狀況：體重減輕程度、飲食狀況、MNA-SF
    - 腸胃道症狀：有無吃不下、噁心、嘔吐或其他腸胃道症狀
    - 憂鬱或精神疾病症狀：GDS 量表
    - 慢性疾病史：心血管、呼吸道、甲狀腺、自體免疫相關症狀
    - 藥物史、使用物質（菸、酒、其他物質濫用）
- PE
    - HEENT：牙齒狀況、是否吞嚥困難 / 說話困難、甲狀腺
    - 神經學檢查
    - 腸胃道、呼吸道、心臟、自體免疫等相關 PE
    - 功能評估：ADL、IADL、get-up-and-go test
- Exam
    - Lab：CBCDC ~(for\ cancer\ or\ anemia)~、CRP / ESR ~(for\ infection/inflammatino)~、BUN / cre ~(for\ CKD)~、electrolytes ~(for\ nutrition)~、AST / ALT / ALP ~(for\ general\ evaluation\ and\ cancer)~、Glu ~(for\ DM)~、LDH ~(for\ malignancy)~、TSH / fT4 ~(for\ thyroid\ disease)~、alb ~(for\ nutrition)~、cholesterol ~(for\ nutrition)、urinalysis ~(for\ CKD\ and\ infection)~、FOBT ~(for\ cancer)~
    - Image：CXR ~(for\ COPD\ and\ cancer)~
    - 其他可考慮檢查：
        - Lab：folic acid、vitamin B12、ferritin、PTH、ACTH、HIV、stool lipid / WBC / parasite
        - Image：腹部超音波、胃鏡 / 大腸鏡、胸部 / 腹部電腦斷層
        - 其他可做或懷疑之癌症篩檢
