# Fatigue 倦怠

### Introduction

- Definition：主觀自覺疲累、無精打采、缺乏活力，導致無法應付日常生活、工作、活動

### Differential diagnosis

| 分類 | 次分類 | 疾病 / 狀況 |
|------|--------|-------------|
| 生理性 | — | 工作、活動、壓力、失眠、休息不足、營養不足、壓力過大、懷孕 |
| 心理性 | — | 憂鬱、慢性焦慮、適應障礙、身體化疾病、纖維肌痛症 |
| 身體性 | 心肺肝腎 | 心衰竭、非典型心絞痛、周邊血管疾病、慢性阻塞性肺病、睡眠呼吸中止、慢性腎病、慢性肝病 |
|        | 感染免疫 | 類風濕性關節炎、紅斑性狼瘡、纖維肌痛症、多發性肌痛風濕、乾燥症、病毒感染、肝炎、結核病、心內膜炎、HIV、CMV、EBV |
|        | 內分泌 | 甲狀腺功能低下或亢進、糖尿病、電解質異常、腎上腺功能不足 |
|        | 血液腫瘤 | 貧血、惡性腫瘤 |
| 藥物及物質 | — | 安眠藥、抗憂鬱藥、酒精、止痛藥、肌肉鬆弛劑、抗組織胺、降血壓藥 |
| 全身勞力不耐症 | — | 全身勞力不耐症 |
| 混合性 | — | 混合性疲勞 |

### Approach

- Hx
    - symptom：duration、onset、diurnal change、related to stress、improved after rest ~(1)~
    - impact：sleep、work ~(1)~
    - associated symptoms：BW loss、fever、夜間盜汗 ~(1)~
    - Past history、drug history、work、travel、life and social ~(1)~
- PE
    - Vital signs：BH、BW、BP、BT、HR ~(1)~
    - HEENT：眼睛、耳朵、鼻咽、LAP、頸靜脈 ~(1)~
    - Chest：heart、breathing sound ~(1)~
    - Abdomen ~(1)~
    - NE：joint、MP、ROM ~(1)~
- Exam
    - 第一階段
        - Blood：CBCDC、CRP、Glu、LFT、RFT、Lipid profile、TP/Alb/Globulin、Na/K/Ca/Mg/P、iron profile、TSH、folic acid、vit B12、HBsAg、anti-HCV ~(1)~
        - Urine：Urine routine、pregnancy test ~(1)~
        - Image：CXR ~(1)~
        - ECG ~(1)~
        - 相關癌症篩檢 ~(1)~
    - 第二階段
        - Blood：LDH、CK、ANA、ESR、RF、Anti-Ro/La、HIV、syphillis、PPD、Lyme disease、drug and alcohol screen ~(1)~
    - 第三階段
        - Blood：cortisol、ACTH、anti-acetylcholine receptor Ab、drug screen、Pb/Cd/Cu ~(1)~
