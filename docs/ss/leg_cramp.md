# Leg cramps 腳抽筋

### Introduction

- Symptoms：最常發生在 posterior calf muscle，但也有可能發生在 foot or thigh ~(2)~
- Epidemiology：高達 60% 的成年人表示曾出現過夜間腿抽筋。F>M ~(2)~
- Pathophysiology：lower motor neuron hyperactivity ~(2)~

### Differential diagnosis

| 分類       | 原因 |
|------------|-------------|
| 肌肉骨骼   | 肌肉疲勞、osteoarthritis ~(2)~ |
| 神經       | Parkinson’s disease、spinal stenosis、radiculopathy、neuropathy、myopathy ~(2)~ |
| 血管       | peripheral vascular disease、chronic venous disease ~(2)~ |
| 慢性病     | liver cirrhosis、ESRD ~(2)~ |
| 藥物       | LABA、beta agonist、beta blocker、thiazide、保鉀利尿劑、ACEi、BZDs... ~(2)~ |
| 其他       | 但仍有許多原因不明 ~(2)~ |


### Approach

- Hx：症狀的發作時間、模式、強度；過去病史；用藥史 ~(2)~
- PE：周邊脈搏~（for\ PAD）~、神經檢查（MP、DTR、sensory、gait）~（for\ neurologic\ disease）~ ~(2)~
- Exam
    - Lab：一般不用，若懷疑神經、血管、慢性病問題，可考慮測 liver function、renal function、B12 ~(for\ neuropathy)~ ~(2)~

### Initial management

- 急性：伸展肌肉 ~(2)~
- 預防：
    - 運動：拉腿後側肌肉 3-5 次，每次 10-20 秒，每天四組。~(2)~
    - 藥物：可考慮 soma、diltiazem、gabapentin、megnesium、verampamil、vitamin B12。但 evidence 不多。~(2)~
