# Fall 跌倒

### Introduction

- 住處內最常發生跌傷的地點：第一為傢俱旁﹙椅子、床、沙發等﹚，佔 23.7％。第二為浴室或廁所等，佔 12.3％。住處外最常發生跌傷的地點，第一為路邊﹙包含人行道﹚，佔 46.2％。第二位為有高度或坡度的地面，佔13％

### Differential diagnosis (Risk factors)

| 分類 | 次分類 | 疾病 / 狀況或影響因素 |
|------|--------|--------------------|
| External | Environment | 地面潮濕、浴廁濕滑、環境雜亂、環境陰暗 |
|          | Wearing | 鞋子或褲子不合、鞋子無止滑 |
| Internal | Neurology | CVA、認知功能受損、意識障礙、帕金森症、前庭障礙、小腦疾病、周邊感覺減退、肌力不足 |
|          | Psychology | 恐懼跌倒、憂鬱 |
|          | Visual | 視力、視野、視覺對比減退 |
|          | Cardiovascular | 姿勢性低血壓、心律不整、心衰竭、周邊動脈疾病 |
|          | Metabolic | 糖尿病、甲狀腺功能高低下、維生素 B12 缺乏、維生素 D 缺乏 |
|          | MSK | 關節炎、骨質疏鬆、足部問題 |
|          | Others | 尿失禁、慢性腎病、慢性肝病 |
|          | Drug | NSAIDs、降血壓藥、利尿劑、抗組織胺、口服降血糖藥、鎮靜劑、抗憂鬱劑、多重用藥（4 種以上） |
| Demographic | — | 年紀大、獨居、過去跌倒、活動量低、ADL 受限、飲酒、女性 |

### Approach

- Hx
    - SPLATT
        - Symptoms
        - Prior fall
        - Location
        - Activity
        - Time
        - Trauma
- PE
    - vital signs、orthostatic HoTN
    - General：鞋子、輔具
    - HEENT：視力、視野
    - CV：arrhythmia、valvular heart disease
    - neuro：mental status、MP、legs sensation、gait、Romberg test、coordination
    - Timed up and go test：3公尺<12秒OK
- Exam
    - Initial
        - Lab：CBC、Chem-7 (BUN, cre, CO2, Glu, Na, K, Cl)、thyroid function、25OH-vitD、drug level
    - Others
        - Heart：event monitor、heart echo
        - Image：brain or spine image
