# Halitosis 口腔異味

### Differential diagnosis

| 分類 | 疾病 / 狀況 |
|------|-------------|
| 口腔原因 | 蛀牙、齒齦炎、牙周病、口腔乾燥、假牙清潔不佳 |
| 咽部 | 扁桃腺結石 |
| 鼻腔 | URI、鼻竇炎、鼻息肉、鼻腔異物 |
| 消化道 | GERD、IBD、gastritis |
| 呼吸道 | bronchiectasis、pneumonia、lung abscess |
| 系統性 | 腎衰竭、肝衰竭、DKA |
| 藥物或化學物質 | 菸、酒、檳榔、disulphiram、cytotoxic agent 等 |

### Approach

- Hx
    - 口腔：清潔狀況、假牙
    - 鼻腔：分泌物、通道是否阻塞、打呼、張口呼吸
    - 系統疾病
- PE
    - 口腔外觀
    - 口腔和鼻腔呼出氣體
    - 硫化物監測器

### Initital management

- 改善口腔衛生：刷牙、牙線、chlorhexidine 漱口水
- 生活習慣：多纖維質、多喝水、少酒少咖啡
