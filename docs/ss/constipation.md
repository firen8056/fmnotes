# Constipation 便祕

### Differential diagnosis

**Primary (functional) constipation**

| 功能性便祕的定義（Rome III criteria） |
|------------------------------------|
| 過去**三個月**有以下**兩個或兩個以上**的症狀，且診斷前六個月以上已出現症狀 |
| 1. 四次中至少有一次或以上的機會需要**費力解大便** |
| 2. 四次中至少有一次或以上的機會**糞便硬** |
| 3. 四次中至少有一次或以上的機會有**解不乾淨的感覺** |
| 4. 有四次之一或以上的機會**感覺肛門堵住** |
| 5. 有四次之一或以上的機會需要**靠肛門周圍或以肛墊方式幫助排便** |
| 6. 每週排便**少於三次** |

- Types of Primary Constipation
    - Normal Transit Constipation (NTC)：最常見，排便次數正常，但仍主觀感覺便秘
    - Slow Transit Constipation (STC)：年輕女性較常見，通常青春期就會發生，排便次數減少（每周小於等於一次）、腹脹腹痛等症狀
    - Defecatory disorder：無法有效排空直腸，有時會需要手挖，常見於老年人

**Secondary**

- Structure
    - 內部狹窄：structure、cancer、anal stenosis、rectocele
    - 外部壓迫
- Systemic
    - Endocrine or metabolic：hyperCa、hypoK、hypoMg、DM、hypothyroidism、hyperparathyroidism、hypopituitary、uremia  
    - CNS：CVA、Parkinson、spinal injury、multiple sclerosis  
    - 腸道神經：Hirschsprung disease、autonomic neuropathy、amyloidosis  
    - Drugs：opioid、anticholinergic、CCB、diuretics、NSAIDs、鐵劑  
- Others
    - IBS-C、飲食生活習慣不佳


### Approach

- Hx
    - 症狀本身：onset、diet、fiber and fluid intake、bowel habit  
    - 合併症狀：有無肛門出血、有無腹瀉、腹痛  
    - 其他病史：thyroid disease、DM、IBS、anorectal disease、electrolytes abnormal、neurologic disease、depression  
    - 藥物、物質史  
- PE
    - Abdomen examination：mass、tenderness、bowel sound  
    - Perineal and rectal：hemorrhoid、scars、fissures、DRE for impaction or stenosis or rectal CA  
- Exam
    - Lab：CBC ~(for\ cancer)~、Glu ~(for\ DM)~、cre ~(for\ uremia)~、Na/K/Ca ~(for\ hypoK/hyperCa)~、TSH ~(for\ hypothyroidism)~、FOBT ~(for\ colon\ cancer)~
    - Image：KUB
    - Others：Coloscopy ~(for\ structure\ evaluation)~

### Initial management

- Bulk-forming（2-3天生效）：Normacol 1pk QD-BID
- Osmotic（2-3天生效）：Forlax 1-2pk QD、Lactulose 15-30ml QD-BID、MgO 1-2# TID-QID、Sorbitol 30g QD
- Stimulant（6-12小時生效）：Sennoside 1-2# HS、Bisacodyl 1-2# HS
- Stool softener：liquid paraffin（適合有NG的用，因為很噁心）、MCT（2-3天一次，每次5-10ml，可到15-20ml，要空腹喝。有些人會腹部絞痛，用完當天-2天後會排便）
- Enema：lactulose 70ml+NS 130ml RC、Evac、Bisacodyl 2#、Glycerin、Fleet enema
- probiotic：要 2-4 周才有效
