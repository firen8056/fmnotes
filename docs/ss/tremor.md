# Tremor 顫抖

### Introduction

- Classification
    - Resting tremor：靜止不動以及有東西支撐時也會tremor（最常見於parkinsonism）
    - Action tremor：動作時的tremor
        - postural tremor：固定姿勢抵抗重力時的tremor，常發生在四肢或頭部
        - kinetic tremor：自主運動時變嚴重。其中有一類叫做intention tremor（goal-directed movement會變嚴重）

### Differential diagnosis

| 分類 | 類型 | 特徵 | 病因 |
|------|------|------|------|
| 生理性 | Enhanced physiologic tremor (P+K) | 姿勢性顫抖，最常見，幅度小、頻率高（10–12Hz） | 交感神經活化（焦慮、恐懼）、內分泌疾病（低血糖、甲狀腺亢進、腎上腺皮質亢進、pheochromocytoma）、藥物、戒斷症狀 |
|  | Essential tremor (P+K) | 以姿勢性為主，也可能為意向性；雙側、4–8Hz；動作時明顯；可影響頭部；隨年齡增加 | 與基因相關 |
| 神經性 | Parkinson disease (R) | 非對稱性靜止性顫抖，可影響上下肢或同側肢體；嚴重時可延伸至動作性顫抖；幅度大於 essential tremor | Basal ganglia 功能異常 |
|  | Cerebellar tremor (K) | 意向性顫抖，幅度大 | 小腦病變（如中風、腫瘤、多發性硬化） |
|  | Neuropathic tremor (P+K) | 與周邊神經病變有關，多為動作性顫抖 | 周邊神經病變（peripheral neuropathy） |
| 其他 | Dystonic tremor (P+K) | 與肌張力障礙相關 | 肌張力異常（dystonia） |
|  | Orthostatic tremor | 少見，站立時下肢顫抖 | 不明 |
|  | Psychogenic tremor (R+P+K) | 可為靜止或動作性顫抖；形式、頻率、方向不固定 | 心因性或壓力相關 |


### Approach

- Hx
    - 症狀本身：幾歲開始、時間變化、何處會抖、加重緩解姿勢
    - 其他可能疾病症狀：甲狀腺亢進症狀、Wilson disease症狀
    - 物質藥物史、加重症狀的物質、工作暴露
- PE
    - Tremor種類：
    手伸直不動（postural）、
    FNF（kinetic）、
    病患靜坐和站著頭下巴舌頭聲音的顫抖（resting）、
    distractability（psychogenic）
    - 神經學症狀：dystonia, rigidity, bradykinesia, cerebellar sign (FNF, heel to shin), gait, pull test
    - 其他疾病：thyroid（凸眼）、Wilson disease（KF ring, hepatomegaly）
- Exam
    - Lab：LFT、thyroid function、血糖（若<40歲可考慮驗血中 ceruloplasmin 及 24 小時尿銅）、毒物及物質濫用篩檢（砷、汞）、pheochromocytoma
    - 需要時再排腦部影像檢查（CT或MRI）
    - 其他檢查包含 polymyographic test（orthostatic tremor）、EMG、TRODAT scan等
    

### Initial management

- physiologic tremor：propranolol
- essential tremor：propranolol 60-320mg in day
- Parkinson、cerebellar、dystonic、orthostatic、psychogenic 等都是治療原本疾病
