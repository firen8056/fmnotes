# Dyspnea 喘

### Introduction

- Definition：Acute dyspnae：< 1個月；chronic dyspnea：> 1個月

### Differential diagnosis

| 系統 | 分類 | 疾病 / 狀況 |
|------|------|-------------|
| Heart | Pump | MI、perimyocarditis、pericardial effusion、valvular heart disease |
|       | Rhythm | arrhythmia |
|       | O₂ capacity | anemia、methemoglobinemia、CO poison |
| Lung  | Airway | asthma、COPD、bronchiectasis、tumor、mucus、foreign body、vocal cord dysfunction |
|       | Parenchyma | edema、pneumonia、atelactasis、ILD |
|       | Vascular | pulmonary embolism、pulmonary hypertension |
|       | Chest wall | pleural effusion、pneumothorax、kyphoscoliosis、increased abdominal girth、ALT、GBS、MG |
| Others | — | metabolic acidosis、hyperthyroidism、anxiety、panic、depression |

### Approach

- Hx
    - 喘本身：time、positional dependence、加重緩解因子、exertion
    - 合併症狀：chest pain、cough、hemoptysis、dysphagia、wheezing、GIB、legs edema、bleeding、thyroid symptoms
    - 其他病史：allergic rhinitis、chest trauma、immunocompromised、anxiety
    - 物質：菸酒、職業暴露 (石綿、有機物、揮發劑、一氧化碳...)
- PE
    - vital signs and fluid status：JVP、edema
    - chest：breathing sound、heart sound
    - skin and MSK：cyanosis、clubbing finger、kyphoscoliosis、漏斗胸、ankylosing spondylitis
    - peripheral pulse
    - Neuro：CN、DTR
- Exam
    - Initial
        - Lab：CBC、chem-8、cardiac enzymes、BNP、D-dimer、lactate、TSH、CRP
        - CXR、ECG、PFT
    - Others
        - Lung：CT、lung echo
        - Heart：heart echo、stress test、holter
