# Leg edema 下肢水腫

### Introduction

- Definition：急性：<72 hrs；慢性：>72 hrs
- Possible pathophysiology
    - change in hydrostatic pressure or oncotic pressure
    - decreased lymph drainage
    - increased capillary permeability

### Differential diagnosis

**單側水腫**

| 分類   | 疾病 / 狀況 |
|--------|-------------|
| 血管   | DVT、venous insufficiency |
| 淋巴   | lymphedema |
| 發炎   | cellulitis |
| 外傷   | muscle or tendon rupture |
| 其他   | Baker cyst、May-Thurner syndrome |

**雙側水腫**

| 分類       | 疾病 / 狀況 |
|------------|-------------|
| 姿勢異常   | 坐或站太久 |
| 系統疾病 (心肝腎) | AKI、CKD、nephrotic syndrome、CHF、cardiomyopathy、pericarditis、pulmonary HTN、cirrhosis |
| 血管       | IVC thrombosis、venous insufficiency、vasculitis |
| 淋巴       | lymphatic disease、lymphedema |
| 藥物       | CCB、steroid、estrogen、hydralazine、TZD、NSAID、docetaxel |
| 其他       | 懷孕、月經前、吃太多鹽、營養不良、自發性水腫 |


### Approach

- Hx
    - 症狀：單雙側、時間
    - 合併症狀：orthopnea、dypsnea、pain、urinary
    - 過去病史：CAD、CHF、HTN、DM、CA、OP、clotting、previous DVT、EtOH、radiation or cath、cellulitis or lymphangitis、mobility
    - 藥物史
- PE
    - HEENT：periborbital edema
    - lung：BS
    - CV：murmur、heart sound、JVP
    - abd：hepatosplenomegaly、ascites
    - lower extremities：circumference、pulses,
- Exam
    - Lab：CBC、BUN/crea、LFT、Alb、BNP、thyroid function、(D-dimer)、Urinalysis
    - Image：CXR、venous duplex、heart echo
