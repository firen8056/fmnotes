# Sexual dysfunction 性功能異常

### Introduction

- erectile dysfunction：無法勃起，或無法維持勃起達到滿意性行為
- premature ejaculation → 常和 ED 一起發生，若同時發生，先治療 ED
    - 正常平均射精時間 5-6 分鐘
    - 原發性：進入陰道後一分鐘內射精
    - 次發性：進入陰道後三分鐘內射精
    - 持續超過半年，75-100% 會發生
- Priapism：疼痛的勃起 > 4 hrs → 要到急診治療
- Dyspareunia：性交疼痛 > 3個月

### Differential diagnosis

**Erectile dysfunction**

| 分類 | 次分類 | 疾病 / 狀況 |
|------|--------|-------------|
| 器質性 | 心血管 | CVD、DM、HTN、hyperlipid |
|        | 神經 | DM、CNS degeneration |
|        | 內分泌 | hypogonadism、hyperprolactin |
| 精神因素 | — | depression |
| 混合性 | — | — |
| 藥物 | — | smoking、alcohol、antiHTN、anticholinergics、antidepressants、anxiolytic、antipsychotics、antiepileptics、antiandrogens、opiates |

### Approach

**Erectile dysfunction**

- Hx
    - 性功能相關：性慾減少、勃起問題（IIEF, EHS）、早洩問題（PEDT）、早晨勃起、自發勃起（可勃起的較少神經血管問題）            
    - 其他症狀：LUTS、genital pain
    - 過去疾病史、藥物史、物質使用史（酒精、非法藥物）
    - 夫妻關係、壓力或緊張、性侵害史
- PE
    - 生殖泌尿：陰莖大小、彎曲、睪丸大小（<20ml 異常）、腹部骨盆腔壓痛腫塊、外陰部感染發炎、DRE
    - 內分泌：第二性徵、視野缺損、乳房（男性女乳）
    - 心血管：股動脈、脈搏
    - 神經：genital and perianal sensation、下肢感覺和肌力、提睪反射
- Exam
    - Lab：CBCDC、肝腎功能、血糖、血脂、FSH、LH、TSH、fT4、AM testosterone (8-10 AM)、E2、prolactin、PSA
    - Image：doppler penile ultrasound（看 PSV、EDV）、血管攝影、陰莖海綿體攝影、Intracavernosal testing with vasoactive substance

### Initial management

**Erectile dysfunction**

- 生活型態調整：戒菸戒酒、減重運動、調整藥物
- PDE5i → intraurethral alprostadil → intracavernous vasoactive drug → VED → penile prosthesis
    - PDE5 inhibitor
        - long QT 不可吃，不可和 nitrate 一起吃（sildenafil 要等 24hrs，tadalafil 要等 48hrs）
        - sildenafil 25-100mg PRN 1hr前吃、vardenafil 5-20mg PRN 0.5-1hr 前吃，空腹吃，一天不可超過一顆
            - cialis 犀利士 24-36hr 較長效
            - viagra 當天使用，1hr 前用
        - tadalafil 5mg QD可以和食物一起吃，定時吃
