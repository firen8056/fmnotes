# Anosmia 嗅覺異常

### Differential diagnosis

| 分類 | 疾病 / 狀況 |
|------|-------------|
| 傳導異常（最常見） | 過敏性鼻炎、鼻竇炎、鼻息肉、鼻中膈彎曲、鼻甲肥厚、鼻腔腫瘤 |
| 神經異常 | 病毒或細菌感染、頭部外傷、額葉腫瘤、嗅神經退化、Alzheimer disease、Parkinson disease |
| 先天性 | 嗅覺發育異常 |
| 新陳代謝疾病 | DM、甲狀腺功能低下 |
| 其他 | malnutrition（vitamin deficiency、pernicious anemia）、Sjogren's syndrome、smoking、toxin、drugs（CCB、beta-blockers、ciprofloxacin、doxycycline、lovastatin、terbinafine、MTX 等） |

### Approach

- Hx
    - 嗅覺：發生時程（突然、漸進、一陣一陣）
    - 合併症狀：頭痛、行為改變
    - 其他病史：職業、抽菸、藥物
- PE
    - 鼻部：鼻腔鼻竇檢查、初步評估嗅覺功能（巧克力、松節油、咖啡、樟腦）
    - 若懷疑CNS問題則做NE
- Exam
    - Lab
        - 可考慮CBC、liver function、renal function、thyroid function
        - 若懷疑營養不良，則測 vitamin
        - 若懷疑自體免疫，則測 ANA、anti-SSA/SSB
        - 若懷疑 toxin，可測 lead、arsenic
    - 其他檢查
        - 嗅覺功能檢查
        - 影像檢查用來評估鼻竇或CNS，通常做CT或MRI
