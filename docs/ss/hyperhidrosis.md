# Hyperhidrosis 多汗

### Introduction

- Definition：benign skin condition of overproduction of sweat beyond the required thermoregulatory response
- Classification
    - Focal：常見於手掌、腳掌、腋下、頭。常因為overactivity of the sudomotor system。常常是 idiopathic，並會被 emotion trigger
    - Generalized：可能跟系統性疾病有關
- 也可分成 primary（idiopathic） or secondary

### Differential diagnosis

| 分類 | 次分類 | 疾病 / 狀況或誘發因子 |
|------|--------|--------------------|
| Primary | Triggers | heat、exercise、anxiety、hot spices、smoking、alcohol |
| Secondary | 生理變化 | menopause、pregnancy |
|            | 心臟 | CHF |
|            | 肺臟 | chronic pulmonary disease |
|            | 內分泌 | DM、hypoglycemia、hyperthyroidism、hypothyroidism、hyperpituitarism |
|            | 慢性發炎 | autoimmune disease |
|            | 慢性感染 | TB |
|            | 惡性腫瘤 | MPN、lymphoma、leukemia、carcinoid tumor、pheochromocytoma |
|            | 神經疾病 | CVA、Parkinson disease、spinal cord injury |
|            | 精神疾病 | GAD |
|            | 藥物物質 | SSRI、SNRI、cholinergic agonist、hypoglycemics、SERM、opioid withdrawal、alcohol use disorder |

### Approach

- Hx
    - Symptoms：age、location、duration、frequency、triggers
    - Other past history or related symptoms
    - Drugs history
- PE
    - Focal skin：maceration、infection
- Exam
    - Focal hyperhidrosis 大部分不用進一步檢查
    - Lab：CBC、Glu、LFT、renal function、ESR、thyroid function、urine catecholamines

### Initial management

- Topical：aluminum salt、topical anticholinergic、topical glycopyrrolate、topical botox
- Systemic：systemic anticholinergic、glycopyrrolate
- botox injection
- Surgery、sympathetic denervation…
