# Chronic cough 慢性咳嗽

### Introduction

- Definition：<3周 → acute cough；3-8周 → subacute cough；>8周 → chronic cough

### Differential diagnosis

| 分類 | 次分類 | 疾病 / 狀況 |
|------|--------|-------------|
| Pulmonary disease | Airway disease | COPD、Cough-variant asthma、Eosinophilic bronchitis |
|                   | Interstitial disease | Idiopathic interstitial pneumonia、Hypersensitive pneumonitis、Pneumoconiosis |
|                   | Infections | Pulmonary tuberculosis、Bronchiectasis、Fungus infection、Cystic fibrosis、Lung abscess |
|                   | Inflammations | Sarcoidosis、Rheumatoid arthritis、Systemic sclerosis、Sjogren syndrome |
|                   | Tumors | Lung tumor、Metastatic tumor |
|                   | Others | Arteriovenous malformation、Recurrent aspiration、Postinfectious cough |
| GI disease | — | GERD |
| Heart disease | — | Heart failure |
| HEENT disease | — | Upper airway cough syndrome (UACS) |
| Drugs or irritants | — | ACEi、Smoking、Inhaled foreign body |

!!! note "Most common etiology"
    UACS、Cough-variant asthma、GERD、ACEi or smoking

### Approach

- Hx
    - Symptom course (onset、duration、triggers)
    - Associated symptoms (hemoptysis、dyspnea、orthopnea、chest pain、PND、sinusitis、hoarseness、reflux、edema、fever、weight loss)
    - Past history (atopy、GERD、CHF、immunocompromise、cancer、TB)
    - Drug and social history (ACEi、smoking、occupation、local epidemiology、TOCC)
- PE
    - Vital signs (include SpO2)
    - HEENT (auditory canal、nasal polyp、pharynx cobblestoning)
    - Lung (breathing sound)
    - Heart (murmur、volume overload、valvular disease)
    - Extremities (clubbing fingers、edema、Raynaud phenomenon、Scleroderma)
- Exam
    - Initial：Chest X ray (PA and left lateral view)、Pulmonary function test (Standard bronchodilator test + impulse oscillometry)、驗痰 (sputum TB and smear x 3 sets) (-by 陳威志醫師)
    - Others
        - Lab：CBCDC、NT-pBNP、sputum stain and culture、sputum AFS and TB culture*3
        - lung：other PFT、chest CT or HRCT、bronchodscopy
            - chest CT：切得較薄，一般看 tumor 用這個 (-by 陳威志醫師)
            - HRCT：解析度較高，但切得較厚，通常用來看 ILD (-by 陳威志醫師)
        - HEENT：sinus image (x-ray or CT)
        - GERD：EGD
