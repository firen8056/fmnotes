# Chest pain 胸痛

### Differential diagnosis

| 分類         | 疾病 / 狀況 |
|--------------|-------------|
| 心臟問題     | angina、ACS、valvular heart disease、pericarditis、myocarditis、pericardial temponade、aortic dissection |
| 肺部問題     | pneumonia、pneumothorax、pulmonary embolism、pulmonary HTN、pleurodynia |
| 腸胃問題     | GERD、esophageal spasm、esophageal rupture、cholecystitis、pancreatitis、PUD |
| 皮膚骨骼肌肉 | shingles、muscular pain、rib fractures、costochondritis、bone mets |
| 其他         | fibromyalgia、rheumatoid disease、psychiatric or anxiety |

### Approach

- Hx
    - 疼痛性質：LQQOPERA  
    - 合併症狀  
        - diaphoresis、dizziness、conscious disturbance、fever  
        - hemoptysis、recent URI、dyspnea、productive cough  
        - tarry stool or bloody stool  
    - Well’s criteria for pulmonary embolism  
    - 共病症：CAD、HTN、DM、hyperlipid、obesity、malignancy、immobility、thrombosis history、recent surgery  
    - 藥物物質：alcohol、smoking、cocaine  
- PE
    - vital signs  
    - CV：heart sound、JVP  
    - chest：breathing sound、chest wall  
    - abd：tenderness  
    - legs：edema、Homans sign  
- Exam
    - Lab：CBC ~(for\ PUD)~、electrolytes、D-dimer ~(for\ PE)~、cardiac enzymes ~(for\ ACS)~
    - CXR ~(for\ lung\ disease)~、ECG ~(for\ ACS)~
    - 可考慮 CT ~(for\ PE\ or\ other\ lung\ disease)~、stress test ~(for\ angina)~、heart echo ~(for\ heart\ disease)~
