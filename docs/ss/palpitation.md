# Palpiation 心悸

### Differential diagnosis

**非心臟相關**

| 分類       | 疾病 / 狀況 | 備註 |
|--------------|-------------|----------|
| Endocrine 內分泌 | hypoglycemia | 盜汗 |
|              | thyrotoxicosis | 失眠、拉肚子、體重下降 |
|              | pheochromocytoma | 頭痛、高血壓、體重下降、高血糖 |
| Fluid 體液     | anemia | 疲倦、蒼白、可能有 pica |
|              | dehydration / Orthostatic hypotension / Postural tachycardia syndrome | 站起來引發 |
| Psychogenic 心因性 | depression | 疲倦、罪惡感、失眠、自殺意念 |
|              | panic / anxiety disorder | 特定因素誘發（如擁擠人群），會有失控感及感覺異常 |
| Other 其他     | 藥物、食物 | 菸、酒、咖啡、茶、麻黃素、腎上腺素、毛地黃、甲狀腺藥物等 |


**心臟相關**

| 次分類   | 疾病 / 狀況 |
|----------|-------------|
| 心律不整 | AT、Afib、Aflutter、AVRT、AVNRT、VT、APC、VPC |
| 瓣膜疾病   | valvular heart disease |

### Approach

- Hx
    - 症狀本身：onset、duration、provocating factors、relieving factors
    - 合併症狀：有無胸悶胸痛、昏厥、頸部搏動
    - 物質：藥物、非法物質、喝酒
    - 其他病史：憂鬱症、焦慮恐慌症篩檢
- PE
    - 心音、看有無心臟衰竭徵象（engorged jugular vein、edema、pulmonary edema）
- Exam
    - Lab：electrolytes (including Mg)
        - 可考慮驗 CBC ~(for\ anemia)~、TSH / fT4 ~(for\ thyrotoxicosis)~、glu ~(for\ hypoglycemia)~、血和尿的 catecholamine ~(for\ pheochromocytoma)~、digoxin level ~(if\ under\ digoxin)~
    - Image：ECG
        - 若有心臟疾病高風險 → Holter ECG → 若陰性再 EP test
            - 高風險
                - 家族史：Sudden cardiac death、CAD、arrhythmia
                - 個人病史：有高血壓、瓣膜性心臟病、冠狀動脈疾病、心肌病變
                - 症狀：有昏厥病史、心悸反覆發生
        - 其他心臟檢查
            - Event monitor recorder、Implantable loop recorder、Exercise treadmill test、EP test
