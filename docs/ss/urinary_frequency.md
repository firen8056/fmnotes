# Urinary frequency 頻尿

### Introduction

- normal bladder capacity：350-650ml，average 400ml
- normal urine frequency：3-7 times per day、1-2 times per night
- frequency：daily urine > 8 times per day or > 1 time every 2 hours
- nocturia：>2 times per night
- urgency：sudden compeling desire to pass urine

### Differential diagnosis

| 主分類 | 次分類 | 疾病 / 狀況 | 說明 |
|--------|--------|-------------|------|
| Polyuria | Osmotic diuresis | hyperglycemia、urea diuresis、sodium diuresis、mannitol | — |
|          | Water diuresis | polydipsia、diabetes insipidus（central、nephrogenic） | — |
|          | Drug | diuretic、alcohol、caffeine | — |
| Decreased bladder capacity | Bladder irritation | UTI、interstitial cystitis、stone、tumor | — |
|                           | Volume decreased | radiotherapy or infection-related fibrosis、surgery、external tumor compression | — |
|                           | Urogenital atrophy | atrophic vaginitis | — |
|                           | Neurodysfunction | stroke、spinal injury、Parkinson disease、multiple sclerosis | — |
|                           | Other | overactive bladder | — |
| Increased residual urine | Obstruction | BPH、stool impaction | — |
|                          | Neurodysfunction | DM、detrusor-sphincter dyssynergy | — |
|                          | Drug | anticholinergic、opioid、α-adrenergic agonist、CCB | — |


### Approach

- Hx
    - 其他LUTS（weak stream, intermittency, straining, empty incomplete, frequency, urgency, nocturia）
    - IPSS score
    - 合併症狀：fever、dysuria、hematuria、膀胱疼痛、漏尿
    - 過去病史、用藥史、手術史或放射線史
- PE
    - 肛門指診
    - 基礎 NE
- Exam
    - 排尿日誌
    - 實驗室：
        - 尿：urinalysis、urine culture、urine cytology、24小時尿量（若>5L驗pitressin）
        - 血：Glu、cre、ADH、PSA（+ free PSA）
    - 其他：
        - 排尿後餘尿、尿路動力學
        - image (KUB、超音波、CT)
        - 膀胱鏡
        - EEG or NCV/EMG、腦部CT
