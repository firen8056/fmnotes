# Headache 頭痛

### Introduction

- Classification
    - primary headache：not due to another cause
    - secondary headache：due to systemic illness or neurological abnormality

### Differential diagnosis

| 分類 | 次分類 | 疾病 / 狀況 |
|------|--------|-------------|
| Primary headache | — | Tension headache、Migraine、Cluster headache |
| Secondary headache | Pressure change | IICP：CNS mass or edema、hydrocephalus、pseudotumor cerebri（類似 IICP，躺下變嚴重）；DICP：CSF leak、over shunting |
|                    | vAscular | stroke、aneurysm or AVM、ICH、EDH、SDH、SAH、venous sinus thrombosis、giant cell arteritis、cervical artery dissection |
|                    | Infection | meningitis |
|                    | Neoplasm | brain tumor |
|                    | Trauma | concussion |
|                    | Other | PRES（posterior reversible encephalopathy syndrome）、TMJ syndrome、glaucoma |


### Approach

- Hx：
    - location：uni、bil、retroorbital
    - quality：pulsutile, tension, stabbing, explosing
    - onset、duration、frequency、time to max intensity
    - trigger factor：外傷、stress, food, touching
    - 誘發緩解因子：姿勢變換, stress, mood, physical strength, noise, chewing
    - associated symptom：噁心嘔吐、頸部疼痛、怕光、aura、暈眩、眼痛、視野缺損、發燒、肌肉疼痛、有無中斷睡眠、其他神經學症狀
    - medication：止痛藥以及使用頻率、鴉片、nitrate、菸酒咖啡
    - PH、FH
- PE
    - vital signs (fever、HTN)
    - face tenderness
    - NE看CN功能、視野+EOM
    - 腦膜刺激症狀
- Exam
    - Lab：若懷疑 CNS ifx 則CBC、CRP、ESR、CSF study。
    - Image：MRI(C+)較好，但如果懷疑急症可到急診做 CT
        - Indication for image → MRI(C+)
            - 神經學症狀：abnormal neuro exam、雙側trigeminal neuralgia或TN + CNV問題
            - 程度嚴重：痛到醒來
            - 症狀特殊：頻率或症狀改變、TAC類的頭痛、活動或Valsava更嚴重、特別的migraine aura、新的頭痛+吐
            - 時間特殊：每天頭痛、>50歲新發生
- Red flag signs → should refer to ER → 先做 CT，加上其他檢查
    - 神經學問題：New focal neuro deficit、GCS change
    - 太嚴重：thunderclap headache (SAH)
    - 特殊的原因：evidence of IICP (mass、venous sinus thrombosis)、meningismus (腦膜刺激表徵)(SAH、meningitis)
    - 特殊族群：immunosupression with fever(CNS ifx)
