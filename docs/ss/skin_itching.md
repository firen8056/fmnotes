# Skin itching 皮膚癢

### Introduction

- Definition：慢性搔癢：> 6周

### Differential diagnosis

**皮膚疾病**

- dermatitis、bullous pemphigoid、urticaria、xerosis、dermatitis herpetiformis、folliculitis、cutaneous T-cell lymphoma、lichen planus、lichen simplex chronicus、psoriasis、pediculosis、scabies

**系統性疾病**

| 分類       | 疾病 |
|------------|------|
| 代謝內分泌 | DM、hyper/hypothyroidism、CKD、carcinoid syndrome |
| 肝膽       | hepatitis、cirrhosis、cholestasis |
| 血液疾病   | IDA、hemochromatosis、leukemia、lymphoma、multiple myeloma |
| 自體免疫   | dermatomyositis、Sjogren syndrome |
| 腫瘤       | solid tumor with paraneoplastic syndrome |
| 感染       | HIV、parasites、prion disease |
| 其他       | drugs、neuropsychiatric disorders、pregnancy |

### Approach

- Hx
    - 症狀本身：location、duration、onset、precipitating factors、weathers
    - 合併症狀：malaise、decreased appetite、body weight change、palpitation
    - 生活狀況：exposure、environment
    - 其他疾病、用藥
- PE
    - focal：skin lesions
    - Other associated disease：jaundice、anemia、thyroid disease、heart rate、nail
- Exam
    - Skin：KOH、Wood's light、皮膚鏡、皮膚切片等
    - Lab
        - 可考慮：CBC ~(for\ hematologic\ disease)~、ferritin ~(for\ IDA)~、Glu ~(for\ DM)~、BUN \ cre ~(for\ uremia)~、TSH ~(for\ thyroid\ disease)~、T-bil \ LFT \ ALP ~(for\ liver\ disease、cholestasis)~
        - 另外也可考慮：ESR ~(for\ autoimmune\ disease)~
    - Image
        - 可考慮 CXR    

### Initial management

- 非藥物
    - 避免濕熱：通風，沖冷水澡、擦汗、穿透氣排汗衣物
    - 保濕：避免過度清潔（消毒水、鹽水、硫磺水會加重癢）、抹乳液乳霜或凡士林
- 藥物
    - topical：steroid、cooling agent、calcinuerin inhibitor、antihistamine or dexepin
    - oral：antihistamine、dexepin、short term steroid
