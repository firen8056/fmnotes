# Dysmenorrhea 經痛

### Introduction

- Classification
    - Primary dysmenorrhea：沒有結構異常，主要和prostaglandin分泌過多有關係
    - Secondary dysmenorrhea：有 underlying 的原因，導致痛經
    - primary v.s secondary
        - 初經不太會痛，通常過了2-3年或大概16歲前後，HPA axis成熟後才比較會痛
        - 若初經就痛，懷疑secondary (結構異常)

### Differential diagnosis

| 分類 | 次分類 | 疾病 / 狀況 |
|------|--------|-------------|
| Primary dysmenorrhea | — | — |
| Secondary dysmenorrhea | Abnormal endometrial tissue | endometriosis、adenomyosis |
|                        | Tumor or mass | leiomyoma、endometrial polyp、adnexal tumor |
|                        | Infection | PID、gonococcal infection、chlamydia infection |
|                        | Structural abnormality | congenital anomalies、mullerian anomalies |
|                        | Others | interstitial cystitis |

### Approach

- Exam
    - Initial：如果像是 primary，則不需要
    - Other：懷疑 secondary
        - Lab：beta-HCG to rule out ectopic or miscarriage、STI test
        - Image：排超音波：腹部 or 經陰道超音波

### Initial management

- 非藥物治療：熱敷，跟ibuprofen相同效果、比acetaminophen效果好
    - 運動、低脂、素食、乳製品、魚油、維生素 BDE 效果不一
- Primary dysmenorrhea
    - 可用 meitifen 75mg QD-Q12H or Naposin 250mg Q12H + buscopan 1tab BID 7 days
    - 無效，可考慮給口服避孕藥。先給 cyclic，無效再改成 continuous or extended
    - 其他治療包含 depot MPA、Vaginal ring、IUD
    - 一直沒有改善，考慮是否為 secondary dysmenorrhea
- Secondary dysmenorrhea（如endometriosis、adenomyosis），則治療原因
