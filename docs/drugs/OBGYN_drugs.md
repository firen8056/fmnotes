# OBGYN clinic common medications 婦產科門診常用藥物

### 荷爾蒙藥物

- 概述：多用來做避孕藥、荷爾蒙補充、調經、止血、產科用。Estrogen 要加上 progestin 來保護子宮內膜  
- HRT  
    - 使用時機：通常是有症狀才使用，強度比避孕藥低，用於更年期症狀或調經。除了 combination 的藥之外，也可用 estromon \+ provera。出血的副作用較 indivina 低。  
    - 禁忌症：心血管高風險、乳癌子宮內膜癌、DVT、肝疾病  
    - 追蹤：定期追蹤乳房攝影或乳房超音波 (可用四癌篩檢 Q2Y 乳攝，中間穿插乳超)  
    - 現在多用 low dose (<50mcg)，風險較低  
    - 通常盡量 5 年內停  
    - 若已經切除子宮，則用 E 就好。  
- OCP：Potency 較 HRT 強，保證濾泡不長大  
    - estrogen 多是 Ethinyl Estradiol，以前劑量較高 (50mcg)，現在 20-35mcg 就夠  
    - progestin 有分很多種，大致可根據 androgen 強度分成四代 (2代\>1代\>3代\>4代)  
    - 禁忌症：心血管高風險 (Hx of MI、CVA)、乳癌子宮內膜癌、DVT、肝疾病、高血壓、\>35歲+抽菸、原因不明陰道出血、(SLE、器官移植)  
    - 追蹤：定期追蹤乳房攝影或乳房超音波 (可用四癌篩檢 Q2Y 乳攝，中間穿插乳超)  

**Estrogen**

| 藥物 | 成分 | 說明 |
| :---- | :---- | :---- |
| Estromon | conjugated estrogen | 馬尿合成的 estrogen，對女性而言，為非生理性雌激素 。thrombosis 風險較高，較常用在 AUB 或調經時使用 |
| Estrade | Estradiol | 結構同於人體分泌的 estradiol，對人體而言為生理性雌激素。通常在不孕科門診比較常使用 |
| Premarin cream | conjugated estrogen | 只擦陰道的 estrogen，主要用來改善停經症候群的生殖泌尿道症狀 |
| Divigel gel | Estradiol | 全身可擦，副作用較口服 estrogen 低。但因為口服其實比較方便，故通常會先用 PO estrogen，無法忍受再改用 divigel。 |

**Progestin**

| 藥物 | 成分及途徑 | 說明 |
| :---- | :---- | :---- |
| Provera | MPA | 有胚胎毒性，因此產科不會用。多用在婦科調經或 AUB 使用。 |
| Utrogestan | progesterone | 多在產科或不孕症穩定子宮內膜時使用 |
| Duphaston | dydrogesterone | 多在產科或不孕症穩定子宮內膜時使用 |

**Combination HRT**

| 藥物 | 成分 | 說明 |
| :---- | :---- | :---- |
| Sevina | estradiol 2mg 12 days estradiol 2mg \+ noresthisterone 1mg 10days estradiol 1mg 6 days | cyclic HRT，通常是還有月經時停經症候群用（因為 cyclic 還可以調經）。在最後 6 天還有低劑量 estrogen，適合無法完全沒有 estrogen 補充的人。 |
| Divina | estradiol valerate 2mg 11 days estradiol valerate 2mg \+ MPA 10mg 10 days | cyclic HRT，通常是還有月經時停經症候群用（因為cyclic 還可以調經）。與 Sevina 的主要差別是最後 6 天無 estrogen。 |
| Indivina | estradiol valerate 1mg \+ MPA 2.5mg 28 days | continuous HRT，通常是無月經之後的停經症候群用 |
| Femonston (自費) | Estradiol 1mg \+Dydrogesterone 5mg | continuous HRT，通常是無月經之後的停經症候群用。研究發現針對預防骨鬆效果好 |
| Livial (自費) | Tibolone 2.5mg 口服後代謝有雌激素、黃體素及雄激素的活性 | continuous HRT，通常是無月經之後的停經症候群用。對乳房影響相對小 |


**Combination OCP**

市面上有很多種，但北榮只有Yasmin，且要自費

| 藥物 | 成分 | 說明 |
| :---- | :---- | :---- |
| Yasmin | Ethinyl Estradiol 30mcg \+ drospirenone 3mg 21 days | continous 的OCP，要自費。drospirenone 是四代 progestin，有anti-androgen 效果，也可抗痘 |

### 泌尿道用藥

**OAB**  

- 效果：vesicare \> oxbu \= detrusitol  
- 副作用：oxbu \> vesicare \> detrusitol

| 藥物 | 機轉 | 說明 |
| :---- | :---- | :---- |
| oxbu | anti-muscarinic | 常是第一線用藥，症狀較不嚴重的 OAB 使用 |
| vesicare | anti-muscarinic(M3) | 症狀較嚴重的 OAB 可用，但較貴 |
| detrusitol | anti-muscarinic |  |
| urotrol | smooth muscle relaxant |  |
| betmiga | beta3-agonist | 通常放在較後線使用。或是 muscarinic antagonist 使用半年後合併用。 |

**尿滯留**

| 藥物 | 機轉 | 說明 |
| :---- | :---- | :---- |
| harnalidge | selective anti-alpha1A |  |
| urief | selective anti-alpha1A | 較 harnalidge 效果更好 |
| bethanechol | cholinergic agonist | 可合併 anti-alpha1A 使用 |

!!!note
    雖然女性沒有 prostate，但膀胱頸的部分仍然有 alpha-receptor，因此 alpha antagonist 對女性尿滯留仍然有效。但要自費使用。
