# Common psychiatric medications 精神科常用藥物

### General concepts

- 憂鬱症劑量 \= GAD  
- OCD 的 SSRI要兩倍劑量  
- Panic 的起始劑量要減半

### Antidepressant

!!!note "副作用"
    - SSRI、SNRI：腸胃不適、睡眠異常、性功能下降的副作用。若與 NSAIDs 併用，GI bleeding 的[風險會上升](https://pubmed.ncbi.nlm.nih.gov/24777151/)。其他藥物使用時機和副作用見表格；與 MAOi 間隔至少需14天
    - TCA：與 MAOi 間隔至少需14天。近期心肌梗塞者不可用、不可和 Dronadarone 同時用。若有 arrhythmia、angle closure glaucoma、可能 AUR、severe liver/renal disease 要審慎使用。

**atypical antipsychotic**

- abilify 副作用較少  
- done 類大部分是第二線   
- pine 類最後一線，易睡易胖  
- 效果：治療正性症狀上，Clozpine較其他藥物更有療效，但因為副作用多，通常當作後線藥物使用。其他藥物沒有明顯差異（效果比較的研究，可參考 [the Lancet 2019](https://www.thelancet.com/action/showPdf?pii=S0140-6736%2819%2931135-3) / [JAMA 2019](https://jamanetwork.com/journals/jamapsychiatry/fullarticle/2725088)）  
- 較不易 QT prolong：Abilify、Zyprexa

- 可合併使用 thiamine 100mg PO QD，folic acid 1mg PO QD，multivitamin
