# Common neurologic medications 神經內科常用藥物

#### 抗癲癇藥 

**依照窄效 v.s 廣效**

- 窄效：Phenobarbital、Phenytoin、Carbamazepine、Oxcarbazepine、Lacosamide、Gabapentin、Pregabalin  
- 廣效：Levetiracetam、Valproate、Topiramate、Lamotrigine、Perampanel、Zonisamide、Clobazam  

**依照機轉**

- GABA↑：BZDs、phenobarbital、Valproate、Topiramate、Vigabatrin、Gabapentin  
- Na channel blocker：Carbamazepine、Oxcarbazepine、Phenytoin、Lamotrigine、Lacosamide、Zonisamide、Topiramate、Valproate  
- Calcium channel blocker  
    - T-type：Valproate、Zonisamide  
    - P/Q type：Gabapentin、Pregabalin  
- Glutamate↓：Perampanel、Topiramate、Valproate  
- SV2A modulation：Levetiracetam

### 巴金森氏症藥物

**For hypokinetic symptoms**

- 種類  
    - Dopamine agonist：Pramipexole、Ropinirole  
        - 使用久了，可能會有 impulse control 和認知功能的問題 (主要是老人)  
    - Levodopa：常搭配 decarboxylase inhibitor 使用  
        - 使用久了，可能會有 motor fluctuation 以及 dyskinesia  
    - Decarboxylase inhibitor（防止 levodopa 在周邊就被代謝）：Carbidopa、Benserazide  
    - MAO-B inhibitor：Selegiline、rasagiline  
        - Contraindication：pheochromocytoma、不可與許多其他藥物（如某些 SSRI、SNRI、bupropion、TCA、MAO inhibitors 等）併用  
    - COMT inhibitor：Entacapone  
    - Amantadine  

**For tremors**

- Anticholinergic：Trihexyphenidyl、Biperiden  
    - Contraindication：narrow-angle glaucoma、urine or bowel obstruction (Biperiden)

### 偏頭痛相關藥物

**急性發作用藥**

- 非專一性藥物：Acetaminophen 1000mg、NSAIDs、aspirin 等  
- Triptans：Sumatriptans (Imigran)  
    - Contraindication  
        - 冠狀動脈疾病：angina、ischemic heart disease（相對ergot類藥物來得安全）  
        - 腦血管疾病 
        - 周邊血管疾病：peripheral vascular disease、Raynaud’s disease  
        - 高血壓控制不良  
        - 肝腎衰竭  
        - 懷孕或哺乳婦女  
        - 使用 ergotamine 24hrs 內不可使用 triptans；使用 MAO-A inhibitors 2周內不可使用 triptans  
    - 健保給付：口服 Sumatriptans 每月不超過 400mg  
- Dihydroergotamine (Lesiton)  
    - Contraindication  
        - 冠狀動脈疾病：angina、ischemic heart disease、history of MI  
        - 腦血管疾病  
        - 周邊血管疾病：peripheral vascular disease、Raynaud’s disease  
        - 高血壓控制不良  
        - 肝腎衰竭  
        - 懷孕或哺乳婦女  
        - 24hrs 內不要併用 serotonin agonist 或 ergot-like agent  

**預防性用藥**

- Indication  
    - 每個月 \> 4次 or \> 8天  
    - 影響生活品質、日常活動  
    - 急性治療藥物失敗  
    - 病患偏好  
    - 特殊的偏頭痛（brainstem aura、hemiplagic migraine、migrainous infarction、prolonged aura 等）

| ----- | ----- | ----- | ----- |
| Anticonvulsant | Topiramate | 50-200 | 會有手麻、體重下降、認知功能下降的副作用 |
|  | Valproate | 300-1500 | 有致畸胎性，懷孕禁用 (很多預防藥物懷孕都不太建議用) |
| Beta-blocker | Propranolol | 20-160 |  |
| CCB | Flunarizine | 5-10 | 會有體重上升、憂鬱。長期使用可能會有 EPS |
| anti-depressant | Venlafaxine | 75-150 |  |
|  | Amitriptyline | 10-75 |  |

**CGRP 單株抗體**

- 藥物：Fremanezumab (Ajovy)、Galcanezumab (Emgality)  
- 健保給付  
    - 需經事前審查核准後使用。  
    - 限神經內科或神經外科專科醫師診斷處方，並不得攜回注射。  
    - 需符合慢性偏頭痛診斷：至少有 3 個月時間，每個月≧ 15 天，每次持續 4 小時以上，且其中符合偏頭痛診斷的發作每個月≧8 天。  
    - 患者需經 3 種 (含) 以上偏頭痛預防用藥物 (依據台灣頭痛學會發表之慢性偏頭痛預防性藥物治療準則之建議用藥，至少包括 topiramate) 治療無顯著療效，或無法忍受其副作用  

**肉毒桿菌**

- 健保給付（同CGRP單株抗體。）  
    - 需經事前審查核准後使用。  
    - 限神經內科或神經外科專科醫師診斷處方，並不得攜回注射。  
    - 需符合慢性偏頭痛診斷：至少有 3 個月時間，每個月≧ 15 天，每次持續 4 小時以上，且其中符合偏頭痛診斷的發作每個月≧8 天。  
    - 患者需經 3 種 (含) 以上偏頭痛預防用藥物 (依據台灣頭痛學會發表之慢性偏頭痛預防性藥物治療準則之建議用藥，至少包括 topiramate) 治療無顯著療效，或無法忍受其副作用
