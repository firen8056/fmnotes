### Medications for tinnitus 

| 商品名 | 學名 | 備註 |
| :---- | :---- | :---- |
| Euclidan | Nicametate | peripheral vasodilator |
| Dogmatyl | Sulpiride | dopamine antagonist |
| Giko | Ginkgoflavone | PLT activating factor antagonist |
| Rivotril | Clonazepam | 有 0.5mg 以及 2mg 的劑型，耳鳴用的是 0.5mg。搭配前三者作安定神經用。 |

!!! note
     若耳鳴，一開始可先用 Euclidan 1# TID + Rivotril 0.5mg 1# HS。若約兩周後無效，改Dogmatyl + Rivotril。若約兩周後無效，再改Giko + Rivotril。
