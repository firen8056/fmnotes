# Common Dermatology medications 皮膚科常用藥物

==以下主要以北榮有的藥物來說明==

### General principle of topical agents
- vehicles：powder、lotion、gel、cream、ointment 等。從清爽到黏膩：lotion > gel > cream > ointment
- quantities：1g cream 可以 cover 10cm^2^ 的皮膚。
    - fingertip unit：大約 0.5 g，可以 cover 雙手的一側。20-30g cream or ointment 可以 cover 成年人全身
- wound care：有很多 discharge 的傷口可以用 ZnO 收斂，或是用 saline 沾濕紗布 wet compression，把 discharge 吸乾
- 擦藥順序：
    - steroid > abx or keratolytic agent > 保濕。每種藥物之間可以隔 5-10 分鐘
    - lotion > gel > cream > ointment，每種藥物間可間隔 5-10 分鐘。若同 vehicle 藥物順序不限。
- Occlusion dressing therapy：可用保鮮膜包住藥膏，可以增加吸收大概 10 倍，另一方面可以避免搔抓。

### Topical antibiotics

| Abx | 使用說明 |
| ---- | ---- |
| Neomycin | 抗菌譜主要為 GNB，大部分抗不到表皮菌，所以基本上當作一條大條藥膏可以隔絕傷口和外界，避免感染的作用。但要注意容易 contact dermatitis。 |
| Tetracycline | 主要抗菌譜為 GPC，但聽說許多表皮菌已經有抗藥性，因此也是當隔絕傷口和外界作用。但因為是眼藥膏，所以比較適合擦在黏膜的部位。 |
| Cleocin (clindamycin) | 可用來治療 GPC 感染以及擦 acne。 |
| Fucidin or conlifu | GPC 首選，但較小條，且較 clindamycin 油 |
| Aczo | 專門拿來擦 acne、folliculitis 等 |

### Other topical agents

**Vitamin D and steroid for psoriasis**

| 種類 | 使用說明 |
| :---- | :---- |
| Daivobet | vitamin D + betamethasone。 |
| Daivonex | vitamin D only，針對不適合擦 steroid 的病人使用。 |

!!! note
    psoriasis 急性期會用 steroid，因為 vit D 擦起來比較刺激。而長期可 nerisone 和 Daivobet 交互擦。

**Calcineurin inhibitor**：可以使用於皮膚發炎病灶。和 steroid 相比較不會造成皮膚萎縮，因此可以長期塗用。

| 種類 | 使用說明 |
| :---- | :---- |
| Tacrolimus | 效果較強。對於小孩，可使用 0.03%。成人則使用 0.1% |
| Pimecrolimus | 效果較溫和，可以給小孩使用 |

**Acne 用藥**

| 種類 | 使用說明 |
| :---- | :---- |
| Adaplene (Differin) | A 酸，可抑制皮脂腺增生，清除毛孔堵塞，殺菌。 |

### Oral antifungal agent

| Antifungal | 使用說明 |
| :---- | :---- |
| Sporanox (Itraconazole) | 對於 Dermatophyte 和 candida 都有效。但肝功能異常以及心臟衰竭患者不能使用。需追蹤肝功能。和其他藥物交互作用多。 |
| Lamisil (Terbinafine) | 只對 Dermatophyte 有效，肝功能異常患者不能用。需追蹤肝功能。 |
| Griseofulvin | 老藥，對於灰指甲治療效果較差，比較常用在頭癬治療上。 |

!!! note
    因為健保針對灰指甲的治療只給付一種 oral antifungal agent，所以有人會先健保使用 Itraconazole (因為比較貴)，若無效再自費使用 Lamisil（另外灰指甲也建議留指甲 culture，確認是甚麼 fungus）
