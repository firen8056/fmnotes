# Common pediatrics medications 兒科常用藥物

### 一般原則

- 12 歲以上或 40kg 以上用成人劑量

### 退燒藥
- Anti-phen：BW/2 ml Q4~6HPRN if BT>38.5
- Ibuprofen：BW/2 ml Q4~6HPRN if BT>39
- Diclofenac RC 1#/12.5kg Q6HPRN (0.5-6mg/kg/day in 2-3 doses)

### 止咳藥
- cough mixture：BW/4 ml TID-QID
- secorine（CTM + pseu + 甘草）：BW/4 ml QID

!!!warning
    brown mixture：小孩禁用!!!

### 化痰化鼻水藥
- Muco syrup：BW/4 ml
- Actein 200mg：BW/40 pck TID-QID：有痰或鼻竇炎也可用，讓鼻水容易流出
- NS inhalation：5ml QID

### 氣管擴張劑
- Meptin：BW/4 ml BID：若聽到 wheezing 時使用
- theophylline：>5 y/o 0.7-0.9ml/kg/dose Q6H (4mg/kg/dose Q6H)、10-16 y/o 200-400mg QD

### 吸入性藥物
- ventolin：<30kg 0.5amp、>30kg 1amp Q4-Q6H，不夠可加 NS 補到至少 3ml，或4-5ml
- atrovent：<12 y/o 0.25-0.5ml、>12 y/o 1ml Q6H  ( 1 amp(>20kg); 0.5 amp(<20kg) in 3ml N/S )
- pulmicort：BW/40 amp BID，但若 asthma 或 croup 可高劑量：0.5-1 amp QID
- combivent：BW/40 amp QID，不夠可加 NS 補到至少 3ml，或4-5ml
- NS inhalation：5ml QID

!!!note
    - 小孩常只吸 ventolin，因為 anticholinergic 會讓痰比較乾，比較難咳出來
    - 蒸氣和化痰藥不要併開，容易被刪

### 氣喘用藥
- Singular：2-5 y/o 4mg QN、6-14 y/o 5mg QN、>=15 y/o 10mg QN。氣喘長期保養用 → 要2周效果才會出來

### 抗組織胺

**一代：小小孩可用，嗜睡，不適合膿鼻涕**

- Cyproheptadine：BW/4 ml TID-QID
- Vistaril：1#/40kg TID-QID
- Spiz (有pseudoephedrine)：BW/5 ml TID-QID

**二代**

- certirizine：BW/4 ml HS：二代，大小孩才能使用
    - 蕁麻疹使用劑量可較高（1.5-4倍）
- Allegra 60mg：1#/40kg BID (1/4#(<2y); 1/2#(2~11y); 1#(>12y))
- IV：chlorpheniramine BW/40 amp IM

### 鼻塞
- pseu solution（pseudoephedrine + cetirizine）：讓鼻黏膜消腫。4 歲以下禁用，但 2-4 歲可考慮減量使用。不可使用超過 7 天 → 會rebond

### 止瀉藥
- hidrasec：<9kg 1pk TID、9-13kg 2pk TID、13-27kg 3pk TID、>27kg 4pk TID
- Smecta：<1y/o 0.33pk TID、1-2 y/o 0.5pk TID、>2 y/o 1pk TID
- Kaopectin：1-2ml/kg/day in TID-QID
- Loperamide：2 歲以上使用：>12kg 1mg ONCE、>20kg 2mg ONCE (兒科盡量少用)

### 促腸胃蠕動、止吐
- Mosapride：1#/40kg/dose TID
- Wempty：1mg/kg/day in TID (BW/4 ml TIDAC)
- Primperan： (BW\*0.1~0.2) mg IM/IV/IF; (BW\*0.1~0.2) mL TIDAC (小心EPS)

### 腸胃抗痙攣藥物
- sesden：1#/40kg/dose
- buscopan：1#/40kg/dose

### 軟便藥

**Osmotic**

- Forlax：>8 y/o用，1-2pk QD
- Normacol：6-12 y/o以上用，0.5-1pk QD-BID
- lactulose：1-2g/kg/day in divided dose (通常給>6 y/o) (BW/2 mL BID)
- Sorbitol：BW/2 - BW/4 ml QD（要自費）
- MgO：BW/40 tab BID; (BW/20 tab TID~QID)

**Stimulant**

- Sennokot：3-6 y/o 0.5# HS、6-12 y/o 1-1.5# HS、>12 y/o 2-3# HS
- Through：3-6 y/o 0.25-0.5# HS、6-12 y/o 0.5-1# HS、>12 y/o 1-2# HS
- Bisacodyl：6-12 y/o 以上用，5-10mg (1-2#) HS
- Evac：<12 y/o 0.5 bot。>12 y/o 1 bot

### 胃藥
- Strocain：1#/40kg/dose TID
- Ranitidine：2mg/kg/dose Q12H，>12 y/o 150mg Q12H or 300mg QD
- Cimetidine：<1 y/o 10mg/kg/dose Q12H，>1 y/o 10mg/kg/dose Q6-12H
- Famotidine：0.5-1mg/kg/day HS or in BID
- Takepron：1mg/kg/day
- Pantoloc：1mg/kg/day PO or IV

### 益生菌
- Biofermin：1-3# TID (BW/20 pack TID~QID)
- Miyarisan：3-6 y/o 0.25pk TID-QID、6-12 y/o 0.5pk TID-QID、>=12 y/o 1pk TID-QID
- Antibiophilus：1-3# TID

### 鎮靜用藥
- Chloral hydrate：0.5mg/kg PO/Rec ONCE，procedure 或短檢查用，30min 前給
- Dormicum：0.1mg/kg IV ONCE

### 抗癲癇藥
- Lorazepam：0.1mg/kg IM or IV ONCE (可 repeat 3 次)
- Diazepam：0.1-0.3mg/kg IV or 0.5mg/kg RC
- Midazolam：0.1-0.2mg/kg IV (max 1mg/kg) (可 repeat 3 次)

### 抗生素
**口服**

- dicloxacillin：25-50mg/kg/day in Q6H
- Amoxicillin：80-100mg/kg/day in TID
- Augmentin PO：BW/2 ml BID (80-90mg/kg/day)
- Ulexin (Keflex)：25-50mg/kg/day (1-2ml/kg/day) in QID
- Cefixime (100mg/cap)：8mg/kg/day in QD-BID
- Azithromycin：BW/4 ml QD (10mg/kg *3 days)

**IV**

- oxacillin：100-150mg/kg/day (severe 150-200mg/kg/day) in Q4-6H
- Ampicillin：100-200mg/kg/day in Q6H
- Augmentin：100-120mg (total)/kg/day in Q8H
- Cefazolin：>1m/o 用，100-150mg/kg/day in Q6-Q8H
- Cefuroxime：150mg/kg/day in Q8H
- cefotaxime：150-200mg/kg/day in Q6-Q8H
- ceftriaxone：50-75mg/kg/day QD
- ceftazidime：50mg/kg/Dose Q8H (100-150mg/kg/day in Q8H)
- Gentamicin：5-7.5mg/kg/day QD

### 流感藥物
- Tamiflu：
    - <12m/o：3mg/kg BID 5D(<3m/o：12mg BID、3-5m/o：20mg BID、6-11m/o：25mg BID)
    - ≥12 m/o：≤ 15kg 30mg BID、16-23kg 45mg BID、24-40kg 60mg BID、≥40kg 75mg BID 5D
- Relenza：≥ 5 y/o才能用。2 puff (10mg) BID 5D
- Rapiacta：>1m/o 可用。10-12mg/kg ONCE，max 300-600mg ONCE (新生兒 6mg/kg)

!!! note
    要症狀開始 48hrs 內給

### 點滴
- D5-0.33%S：<20kg 或 <8歲
- D5-0.45%S：>20kg 或 >8歲

### 眼藥水
- Zaditen：抗組織胺。1 gtt BID：For 過敏性結膜炎。
