!!!note
    有防腐劑的 eyedrop 一天不可點超過 4 次，若乾眼仍嚴重，可加 gel，一天可到 TID or QID。若更嚴重，則 gel 換 ointment

### 抗生素眼藥水/藥膏

| 劑型 | 學名 | 商品名 | 前房 penetration | 備註 |
| ----- | ----- | ----- | :---: | ----- |
| drop | Chloramphenicol | Chloramphenicol "S" eye\*drop 5ml | \- | 要冰，較不方便，但點起來較舒服 |
|  | Sulfamethoxazole | Sulmezole eye \* drops 4% 10 ml | \- | 不用冰，較大瓶。但要小心過敏。 |
|  | Levofloxacin | Cravit oph sol 0.5% 5 ml | V |  |
|  | Ciprofloxacin | Ciloxan eye soln 0.3% 5ml | V | 北榮無 |
| oint | Gentamicin | GENDERMIN OPH \* OINT 3 MG/G 5 G | \- |  |

!!!note
    chloramphenicol、sulmezole、gentamicin 大部分效果普通，大概是消除表面 normal flora，避免次發細菌感染用。如果真的細菌感染，還是要用Cravit 或其他 fluoroquinolone（如 moxifloxacin, 即 Vigamox 可至川秀/維康藥局買），或會依細菌藥敏測試選擇抗生素（但需自己泡藥）。若是黴菌感染，可泡 ampB/voriconazole/市售的 natamycin.
